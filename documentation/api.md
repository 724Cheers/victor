## add(vector,)

Adds another vector to this one

### Examples:
    var vec = new Victor(10, 10);
    var vec2 = new Victor(20, 30);

    vec.add(vec2);
    vec.toString();
    // => x:30, y:40

### Params: 

* **Vector** *vector,* the other vector you want to this one

### Return:

* **Vector** this, for chaining capabilities
